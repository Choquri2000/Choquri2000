Data Analyst by trade, Data Scientist by ambition. I started in BI and engineering — building dashboards, automating pipelines, designing warehouses. Somewhere along the way I caught the ML bug. Now every project I touch gets a prediction model, a statistical test, or at least a well-tuned hyperparameter.
